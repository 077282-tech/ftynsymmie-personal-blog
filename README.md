# Fatin E-Portfolio

## Project Title

Fatin E-Portfolio

---

## Description

Fatin E-Portfolio is a personal portfolio website developed for the CSD34203 Personal Blog Project. This website showcases my personal information, educational background, projects, experiences, skills, business activities, club involvement, and contact information.

---

## Features

- Home Page
- About Me Page
- Education Information
- Personal Strengths
- Career Goals
- Business Experience
- Club Involvement
- Portfolio Showcase
- VR Project
- Photography Gallery
- Videography Portfolio
- Graphic Design Portfolio
- 3D Animation Portfolio
- Contact Information

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages
